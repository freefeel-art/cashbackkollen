# Varför registreras inte min cashback hos Refunder?

Ett av de vanligaste problemen som cashback-användare stöter på är att ett köp inte registreras trots att köpet har genomförts korrekt.

Det kan vara frustrerande att vänta på cashback som aldrig dyker upp, men i de flesta fall finns det en förklaring.

I den här guiden går vi igenom de vanligaste orsakerna till att cashback inte registreras hos Refunder och vad du kan göra för att minska risken för problem.

## Kort svar

När cashback inte registreras beror det vanligtvis på att spårningen av köpet har misslyckats.

Det betyder inte nödvändigtvis att något är fel på Refunder eller butiken.

Vanliga orsaker är:

- Adblockers
- Blockerade cookies
- Externa rabattkoder
- Byte av enhet under köpet
- Flera webbplatser före köpet
- Tekniska problem i spårningen

## Hur registreras cashback?

När du klickar vidare till en butik via Refunder registreras information som gör det möjligt att koppla köpet till ditt konto.

Om allt fungerar korrekt får Refunder information om köpet och cashback kan registreras.

Om någon del av processen bryts kan spårningen misslyckas.

För en fullständig genomgång av processen kan du läsa vår guide om [hur Refunder fungerar](/hur-fungerar-refunder/).

## Vanligaste orsakerna

### Adblockers

Annonsblockerare kan ibland blockera den teknik som används för att registrera köp.

Det kan göra att butiken inte kan koppla köpet till Refunder.

Om du använder adblocker kan det vara klokt att tillfälligt stänga av den innan köpet genomförs.

### Blockerade cookies

Cookies används ofta för att hålla reda på vilken länk som ledde till köpet.

Om cookies blockeras eller raderas kan registreringen misslyckas.

### Externa rabattkoder

Många användare vill kombinera cashback med rabattkoder från andra webbplatser.

Det kan ibland påverka spårningen.

I vissa fall kan butiken istället tilldela provisionen till den webbplats som tillhandahöll rabattkoden.

### Flera cashback-sajter

Om du besöker flera cashback- eller jämförelsesidor innan köpet kan det bli oklart vilken aktör som ska få provisionen.

Detta kan påverka registreringen.

### Byte av enhet

Om du klickar på Refunder-länken på en mobiltelefon men genomför köpet på en dator kan spårningen brytas.

Det säkraste är att genomföra hela köpet på samma enhet.

### För lång väntetid

Om du väntar länge mellan klicket och själva köpet kan registreringen i vissa fall förloras.

## Tekniska problem

Även om du gör allting rätt kan tekniska problem ibland uppstå.

Exempel:

- Problem hos butiken
- Problem hos affiliate-nätverket
- Problem i rapporteringen mellan systemen

Det är ovanligt men förekommer.

## Hur minskar man risken?

Följande rutiner kan öka chansen att cashback registreras korrekt.

### Logga in först

Se till att vara inloggad på Refunder innan du klickar vidare.

### Klicka direkt till butiken

Undvik att besöka andra webbplatser innan köpet slutförs.

### Slutför köpet direkt

Genomför köpet i samma session.

### Använd samma enhet

Byt inte mellan mobil, surfplatta och dator.

### Läs butikens villkor

Olika butiker har olika regler för cashback.

## Om cashback fortfarande saknas

Om köpet inte registreras trots att du följt rekommendationerna bör du:

- Spara orderbekräftelsen
- Dokumentera köpdatum
- Spara kvitton och ordernummer
- Följa Refunders rutiner för supportärenden

Ju mer information du har kvar desto lättare blir det att undersöka vad som hänt.

Du kan även läsa om [hur lång tid det tar att få cashback hos Refunder](/hur-lang-tid-tar-det-att-fa-cashback-hos-refunder/) eftersom många användare tror att cashback saknas när den egentligen fortfarande väntar på verifiering.

## Är utebliven cashback ett tecken på att Refunder är osäkert?

Normalt inte.

De flesta problem handlar om spårning och inte om säkerhet.

Cashback bygger på att flera tekniska system kommunicerar med varandra och ibland uppstår fel i processen.

Läs mer i vår guide om [huruvida Refunder är säkert](/ar-refunder-sakert/).

## Vanliga frågor

### Registreras cashback alltid direkt?

Nej. Det kan ta allt från några timmar till flera dagar innan ett köp syns.

### Kan rabattkoder påverka cashback?

Ja. Vissa rabattkoder kan göra att cashback inte godkänns.

### Kan adblockers påverka registreringen?

Ja. De kan ibland störa spårningen.

### Är det vanligt att cashback inte registreras?

Det händer ibland, men de flesta köp registreras korrekt när villkoren följs.

## Sammanfattning

När cashback inte registreras hos Refunder beror det oftast på att spårningen av köpet har misslyckats. De vanligaste orsakerna är adblockers, cookies, externa rabattkoder eller att köpet genomförts på ett sätt som bryter spårningen.

Genom att klicka direkt via Refunder, använda samma enhet och följa butikens villkor kan du minska risken för att cashback uteblir.
